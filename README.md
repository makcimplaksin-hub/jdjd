Квадратная матрица - операции на C++

Программа работает с квадратной матрицей, реализованной на C++. Предоставляет функции:

енерация случайной матрицы n×n с элементами от 1 до 100.
Вычисление суммы элементов главной диагонали.
Поиск и обмен максимальных элементов, расположенных выше и ниже главной диагонали.
Вычитание минимального элемента матрицы из всех элементов столбцов, расположенных справа от столбца с этим минимальным элементом.

Функционал

1.Создание матрицы
   Заполняется случайными целыми числами от 1 до 100.

2. Сумма главной диагонали
   Вывод суммы элементов, расположенных на главной диагонали матрицы.

3. Обмен максимальных элементов выше и ниже главной диагонали
   Находит максимальные элементы, расположенные строго выше и строго ниже главной диагонали, и меняет их местами.

4.Корректировка столбцов по минимальному элементу 
   Определяет минимальный элемент в матрице, находит столбец с ним, затем вычитает это значение из всех элементов столбцов, идущих правее.

---

## Запуск и использование

1. Скомпилируйте программу с помощью компилятора C++ 

2. Введите размер квадратной матрицы.

3. В меню выберите нужную операцию, следуя подсказкам.

4. Для выхода выберите пункт 0.

Требования

Компилятор C++11 и выше
Консоль для ввода и вывода данных

Пример

Введите размер квадратной матрицы: 4

Меню:

1 - Сумма элементов главной диагонали

2 - Поменять местами максимальные элементы выше и ниже главной диагонали

3 - Из элементов столбцов, расположенных после столбца с минимальным элементом, вычесть этот минимальный элемент

0 - Выйти

Выберите пункт: 1

Сумма элементов главной диагонали: 215
